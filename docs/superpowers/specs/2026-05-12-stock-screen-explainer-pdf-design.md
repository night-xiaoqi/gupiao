# 股票截图解释型 PDF 设计说明

## 1. 目标
基于图片素材“东方财富.jpg”，生成一份对照标注版 PDF。成品不再采用教师讲义思路，而是围绕截图中实际可见的数据与按钮，分区域解释它们分别代表什么，并补充少量股票软件常识，帮助用户直接看懂这张图。

## 2. 成品范围
- 输出格式：PDF
- 内容形式：对照标注版
- 主要素材：东方财富.jpg
- 说明风格：白话 + 一点股票常识
- 解释范围：以截图中看得到的数据和按钮为主，并顺带说明它们在炒股软件里通常代表什么

## 3. 组织方式
采用“按区域分块解释”的方式，而不是简单按截图从上到下罗列。

预计拆分为以下区域：
1. 顶部股票基本信息区
2. 价格与涨跌区
3. 估值与资金区
4. 关联信息与功能切换区
5. 分时 / 五日 / 日K 等图表切换区
6. 分时图主图区域
7. 成交量区域
8. 底部功能区

## 4. 页面结构
预计 PDF 采用 2–4 页结构：

### 4.1 封面/说明页
- 标题
- 原图缩略展示
- 说明这份 PDF 的用途：帮助读懂截图中的每项数据

### 4.2 主体解释页
主体页按区域拆分，每个区域包含：
- 区域标题
- 区域截图或原图局部
- 该区域中的数据项名称
- 每个数据项的白话解释
- 必要时补充“通常怎么看”这一类简短说明

### 4.3 汇总页
- 把最关键的数据项再集中列一遍
- 便于快速回顾

## 5. 内容风格
### 5.1 解释方式
- 优先用白话说明“这是什么”
- 再补一句“通常怎么看”或“在软件里一般表示什么”
- 不展开到投资建议层面
- 不对买卖做推荐

### 5.2 解释深度
解释以“看懂界面”为目标，不延伸成系统证券课程。

## 6. 具体解释对象
应优先覆盖截图中明显可见的信息，例如：
- 股票名称与代码
- 所属市场标签
- AI 按钮、搜索、分享等功能按钮
- 当前价格
- 今开、最高、最低
- 涨跌额、涨跌幅
- 换手率
- 总手、金额
- 总值、流值、市盈率
- 相关基金、持仓占比
- 分时、五日、日K、周K、月K等切换项
- 均价、最新、图右侧百分比刻度
- 成交量、分时量、现量
- 底部导航与自选按钮

## 7. 排版原则
- 尽量保持原图可辨识
- 说明文字与图片局部一一对应
- 每页信息量适中，避免拥挤
- 重点数据使用编号或项目符号区分
- 整体风格以清晰、实用为主，不做讲义化包装

## 8. 边界与限制
- 只解释截图中能看到或能合理识别的内容
- 若截图中部分数字过小或模糊，可结合常见股票软件字段进行谨慎说明，但不得捏造具体不可见数据
- 对功能按钮只解释常见用途，不假设用户的具体使用场景

## 9. 生成策略
推荐实现路径：
1. 识别截图中的主要区域与可见字段
2. 为每个区域准备对应解释文案
3. 用可打印的版式把原图、局部图和解释排成多页 PDF
4. 输出最终 PDF 文件

## 10. 验证标准
交付前应确认：
- PDF 成功生成
- PDF 中确实使用原图进行对照解释
- 内容按区域分块
- 解释风格为白话 + 一点股票常识
- 覆盖截图中的核心数据项与功能区
- 排版清晰可读
